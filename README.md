# HFPL AI Assistant
### Hyper Filteration Pvt. Ltd. — Official AI Brand Agent

An AI-powered chat assistant that promotes HFPL's water treatment EPC services across the web, WhatsApp, and AI platforms.

**Live URL:** https://hyperfilteration.github.io/ai-agent/

---

## What It Does
- Answers queries about ZLD, ETP, STP, RO-EDI, CETP, MBR, MEE/ATFD
- Showcases HFPL's key projects (Nakkapally, Una, Ionex, RTM, PM MITRA)
- Guides visitors on Indian environmental compliance (CPCB, ZLD mandates)
- Qualifies leads and drives proposal requests
- Powered by Claude AI (Anthropic)

## Tech Stack
- Single HTML file — no backend needed
- Claude claude-sonnet-4-20250514 via Anthropic API
- Deployed via GitHub Pages (free hosting)

## Deploy
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source: main branch / root**
3. Your URL: `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Embed on Website
```html
<iframe src="https://hyperfilteration.github.io/ai-agent/" 
  width="100%" height="700px" frameborder="0" 
  style="border-radius:12px;border:1px solid #d4e8df;">
</iframe>
```
